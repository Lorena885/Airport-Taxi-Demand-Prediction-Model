What is CatBoost?

CatBoost is a gradient boosting machine learning algorithm that combines many decision trees to produce accurate predictions. It is especially useful for structured data and can handle different types of features efficiently. In this project, CatBoost was used to forecast hourly taxi demand based on historical order patterns.
